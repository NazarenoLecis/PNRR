# Metodologia

Il repository misura la distribuzione territoriale e l'avanzamento finanziario dei progetti PNRR usando dati aperti.

I pagamenti sono trattati come stock cumulati alla data di aggiornamento del dataset. Per analisi temporali servono snapshot periodici.

I progetti multi-localizzati sono ripartiti in modo uniforme entro ciascun livello territoriale. Se un progetto è associato a tre comuni, ogni comune riceve un terzo dell'importo negli aggregati comunali.
